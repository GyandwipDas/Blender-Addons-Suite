How to use

1. Download the zip file
2. Install it by going into Edit > Preferences > Addons / Get Addons > Install from Disk >
3. Once you open you lighting file go to tools side panel(N shortcut on keyboad) scroll to Duplicate Light Remover Panel and select which preset you want
4. Can change these values by going into C:\Users\<user>\AppData\Roaming\Blender Foundation\Blender\<whichever blender version you are using>\scripts\addons
