**How to use**

1. Download the zip file
2. Install it by going into Edit > Preferences > Addons / Get Addons > Install from Disk > <Pick the downloaded zip>
3. Once you open you lighting file go to tools side panel(N shortcut on keyboad) scroll to Duplicate Light Remover Panel and delete
